# 🔤 Guess the Word

**Guess the Word** é um jogo interativo onde o jogador deve adivinhar uma palavra secreta, letra por letra, com base em uma dica fornecida.

## 🎮 Sobre o Jogo

O objetivo é descobrir a palavra correta antes de atingir o limite de erros. A cada letra correta, o jogador acumula pontos, tornando o jogo mais desafiador e envolvente.

Com três erros, a partida é encerrada — mas você pode reiniciar e tentar novamente para melhorar sua pontuação.

## 🛠️ Tecnologias Utilizadas

- React
- JavaScript
- HTML
- CSS

## ⚙️ Funcionalidades

- Sistema de adivinhação por letras
- Sistema de pontuação
- Controle de erros (limite de tentativas)
- Reinício da partida

## 🧠 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento com **React**, incluindo gerenciamento de estado, componentes e interação com o usuário.
